# Amazon Sales Dashboard using Excel

## Project Overview

This project is an interactive **E-Commerce Sales Dashboard** created using Microsoft Excel. After building my previous **Bank Loan Report Dashboard**, I worked on this project to explore another business domain and improve my dashboarding and data analysis skills.

The dashboard helps analyze e-commerce sales performance by tracking important business metrics such as total revenue, total orders, quantity sold, average order value, and delivered order percentage.

## Dashboard Features

* KPI Cards for key business metrics
* Monthly Sales Trend analysis
* Sales by Product Category
* Order Status Distribution
* Payment Method Analysis
* Interactive slicers for Year, Category, Order Status, and Country
* Clean and professional dashboard layout

## Key KPIs Tracked

* Total Revenue
* Total Orders
* Total Quantity Sold
* Average Order Value
* Delivered Orders Percentage

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards
* Data Cleaning and Formatting

## Key Learnings

Through this project, I learned how to convert raw transactional data into a meaningful and interactive dashboard. I improved my understanding of Pivot Tables, Pivot Charts, slicers, KPI calculation, and dashboard design.

This project also helped me understand how Excel dashboards can be used to support business decision-making by presenting data in a clear and structured way.

## Conclusion

This dashboard is part of my learning journey in **Data Analytics, Microsoft Excel, and Business Intelligence**. It helped me practice real-world data analysis and improve my ability to present insights visually.
